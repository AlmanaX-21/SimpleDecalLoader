# SimpleDecalLoader

A BepInEx plugin for Aviassembly that allows loading custom decals. Simply upload any image to load into the game.

## Features

- Load custom decals from `BepInEx/config/Decals`.
- Reload decals in-game with a button.
- Logs button clicks for debugging.

## Installation

1. Install BepInEx for Aviassembly.
2. Place `SimpleDecalLoader.dll` in the `BepInEx/plugins` folder.
3. Launch the game once to allow `decals` folder to generate.
4. Now you can put in any png image to load it into the game.
